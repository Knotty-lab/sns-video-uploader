Privacy Policy

← Quiz Video Uploader

Effective date: July 10, 2026

Quiz Video Uploader ("the tool") is a personal, single-user application operated by an individual developer ("the operator"). The operator is the only user of the tool. This policy describes what data the tool accesses and how it is handled.

1. Data the tool accesses

  • The operator's own account information (channel/account ID, username, avatar) — used to display which account an upload will go to and to perform the upload.
  • The operator's own video files and metadata (titles, descriptions, hashtags) — created locally by the operator and uploaded to the operator's own accounts.
  • OAuth access tokens issued by each platform after the operator signs in.

The tool does not access, collect, store, or process any data belonging to third parties, viewers, or other creators. It does not use cookies, analytics, or advertising.

2. YouTube API Services

The tool uses YouTube API Services to upload videos to the operator's own YouTube channel. By using the tool, the operator agrees to be bound by the YouTube Terms of Service. Google's handling of data is described in the Google Privacy Policy.

The tool accesses only the authenticated operator's own channel through the scopes the operator grants. Authorized data (OAuth tokens) is stored only on the operator's local machine and is never transmitted to any party other than Google/YouTube. The operator can revoke the tool's access at any time via Google security settings.

3. TikTok and Instagram APIs

The tool uses the TikTok Content Posting API and the Instagram Graph API in the same manner: to upload the operator's own videos to the operator's own accounts. Access tokens are stored only on the operator's local machine. Access can be revoked at any time from each platform's connected-apps / security settings.

4. Data storage, retention, and deletion

  • All data handled by the tool (video files, metadata, OAuth tokens) resides solely on the operator's local machine. No server operated by the tool exists, and no data is shared with or sold to any third party.
  • OAuth tokens are retained only while the tool is in use and are deleted by removing the local credential files, or invalidated by revoking access on each platform.
  • Locally stored video files and metadata are deleted by the operator as part of normal file management; nothing is retained outside the operator's machine.
  • Because the tool has no users other than the operator and stores no third-party data, there is no user data to request deletion of. Any inquiry can be sent to the contact below and will be answered within 7 days.

5. Changes to this policy

If the tool's behavior changes in a way that affects data handling, this page will be updated before the change takes effect.
